# 下面是让ai写的（我自己也没试效果咋样，反正能接通）：
这是一个为 OpenClaw 设计的专家级扩展插件。

### 🌟 核心功能
- **智能外援**：当 OpenClaw 本地模型逻辑“卡壳”时，一键求助 Gemini 2.5 Pro。
- **自动汇总**：自动提取上下文、代码段和报错信息，生成“案情简报”发给专家。
- **网络穿透**：内置代理配置，完美解决国内环境下 API 连接问题。

### 🛠️ 如何安装
1. 下载本项目，将文件夹重命名为 `gemini_expert`。
2. 将文件夹放入 OpenClaw 的 `skills` 目录。
3. 安装依赖：`pip install google-genai`。
4. 在 `ask_gemini.py` 中填入你的 API Key 和代理端口。
